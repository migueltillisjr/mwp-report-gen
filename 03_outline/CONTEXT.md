# Stage 3 — Outline

Build a structured outline with introduction, body sections, and conclusion.

## Input

Read `02_research_and_sources/output/sources.md` and `02_research_and_sources/output/frameworks.md`.

## Process

Build a structured outline:
- **Introduction** — hook, context, thesis statement
- **Background** — define key terms and concepts
- **Analysis** — apply relevant frameworks to the topic (multiple perspectives)
- **Argument** — state and defend the position
- **Counterarguments** — address objections fairly
- **Conclusion** — summarize, restate thesis, broader implications

Present the outline and confirm before moving on.

## Output

Write the following to `03_outline/output/`:

| File | Contents |
|------|----------|
| `outline.md` | The confirmed structured outline with section headings and bullet points |

## Run Versioning

On subsequent runs, append a unique run ID (timestamp) to the filename:
- First run: `outline.md`
- Subsequent runs: `outline_<YYYYMMDD_HHMMSS>.md`

Generate the timestamp using `bash`: `date +%Y%m%d_%H%M%S`
