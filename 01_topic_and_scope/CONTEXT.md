# Stage 1 — Topic & Scope

The topic, scope, and assignment requirements are in `shared/references/`. Read `REQUIREMENTS.md` and `RUBRIC.md` there for full context.

## Process

Ask the user:
1. What is the required length and citation style?

Once confirmed, move to Stage 2.

## Output

Write the following to `01_topic_and_scope/output/`:

| File | Contents |
|------|----------|
| `scope.md` | The topic, scope, thesis direction, required length, and citation style |

## Run Versioning

On subsequent runs, append a unique run ID (timestamp) to the filename:
- First run: `scope.md`
- Subsequent runs: `scope_<YYYYMMDD_HHMMSS>.md`

Generate the timestamp using `bash`: `date +%Y%m%d_%H%M%S`
