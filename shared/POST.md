I built a multi-agent system that generated reports in 1 day using OpenCode and Amazon Web Services (AWS) models.

I've learned to change my mindset and, instead of focusing on prioritizing frameworks, I've decided to build with the best technologies that deliver the greatest outcomes for my vision.


No programming frameworks.
No LangChain.
No LangGraph.
No PydanticAI.

Just Markdown .md files, coding agents, and runtime agents working together through a simple, structured process.

The idea is straightforward:

Use Markdown as the control layer.
Use agents as the execution layer.
Use local runtime behavior to keep the system flexible, observable, and easy to change.

It is still a work in progress, but even in its current state, it gets the job done.
